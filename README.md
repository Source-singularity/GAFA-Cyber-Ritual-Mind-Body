## 📖 核心立意：意智与异质的边界拷问 (Artistic Statement)

血、肉、指甲、头发，是具体的物，组成人体的客观实在的物；而“意智”是客观存在的思维。两者看似截然不同，但在“人”处实现统一，形成客观的人。那么在“人”处，意智和异质联系的强度是多少？容量是多少？这就成为了一个必须被摆上台面研究的问题。

现实中，咬指甲、卷头发、在意脸上的痘痘……这些意智对异质呈现出具体的、普遍性的自动反应。这些反应的差异成为了一个切入点：**将潜意识的自动行为，变成思辨的分析，进而控制特定意智与异质组合的联系。** 实际上，这也是对“人的边界”的探讨，契合马克思主义哲学对人解构的范畴（精神与物质）。

对这类联系的思考自古有之（如残疾人假肢的“机械感”与“仿生感”之争）。而今天，我们面临的是人机交融的明天：从被使用的工具，到外骨骼，再到脑机接口。面对这种技术奇点，我们如何引导选择？

本项目从当代年轻人极具时代特征的**“钉”文化（人体穿孔）**入手。这个“钉”不再是如 Orlan（欧兰）的面部改造、或是 Stelarc（斯蒂拉克）在手臂上种耳朵这种基于本体论对身体异质功能的拓展；而是在这种拓展正在发生的背景下，使“意智态度”亮相。
当机械、算法乃至于客体的艺术家纷纷呈现，引导一种社会“景观”时，作为普罗大众更需要走出潮流，以主体视角思考风云变幻。

**本作品是一个微型艺术品、一个“测谎仪式”、一座祭坛。**
它呈现年轻人往身上穿孔时，异物穿过伤口抽动的情境。通过制造一个祭祀现场，参与者成为了祭司。在祭祀的过程中，强化了参与者的主体性，削弱了社会景观对意智的规训。被再现的异质与意智的矛盾，或能激发具体人对自己所适用的关系的思考，捍卫其在后人类时代的主体性。

## ⚙️ 系统架构与核心技术 (System Architecture)

本项目由 **PC上位机（AI视觉推理）** 与 **STM32下位机（硬件实时控制）** 协同构成。为了保证极端的机械丝滑感，下位机采用了**工业级防撕裂状态机**与**底层寄存器中断驱动**。

1. **多模态输入层**：
   * **面部情绪捕捉**：PC 端调用 RTX GPU 运行 HSEmotion 模型，实时推断 8 种微表情。
   * **生理心率监听**：手指贴合 MAX30105 传感器，STM32 边缘解算脉搏波形与实时 BPM。
2. **硬核决策层 (STM32 非阻塞原子操作)**：
   * **原子级状态机**：利用 `noInterrupts()` 防止 ISR 中断介入导致的数据撕裂，确保极高频的电机脉冲稳定下发。
   * **幽灵记忆机制**：当参与者因不适移开手指时，系统会锁死在最后一次读取的生理节律中持续运转。
3. **物理输出层**：
   * **穿刺模拟 (步进电机)**：使用底层 `TIM2->ARR` 寄存器重载频率，完全绕过高层库的延迟，精准控制 10000 个高频脉冲，完成 10cm 行程的暴躁穿刺。
   * **抽动模拟 (电磁铁)**：精准捕捉每一次脉搏波峰（`diff < 0`），驱动 12V 电磁铁产生 100ms 的短促强烈撞击，模拟伴随心跳的抽动痉挛。

### 📡 紧凑型串口协议 (Communication Protocol)
* **下位机至上位机 (STM32 → PC)**：`<W:波形值,B:BPM值,E:情绪>` (例: `<W:45.2,B:72.0,E:neutral>`)
* **上位机至下位机 (PC → STM32)**：`<EMO:情绪标签>` (例: `<EMO:anger>`)

## 🛒 硬件清单与接线指南 (Hardware Setup)

⚠️ **危险警告：接线时务必断开 12V/24V 总电源！带电插拔电机或电磁铁会导致驱动板瞬间炸毁！**

### 1. 核心材料 BOM
* **主控**：STM32F103C8T6 (BluePill)
* **传感器**：MAX30105 / MAX30102 心率血氧模块
* **穿刺机构**：42步进电机 (1.5A) + TB6600 驱动器 + T8 丝杆 (导程 8mm)
* **抽动机构**：12V 推拉式电磁铁 (行程 10-12mm) + **MOS管驱动模块 (如 IRF540N)**
* **电源**：12V 3A (或以上) 直流电源适配器

### 2. 步进电机接线 (共阳极 5V 逻辑)
* `TB6600 PUL+`, `DIR+`, `EN+` ➔ **全部短接** 并连到 STM32 `5V`
* `TB6600 PUL-` ➔ STM32 `PB12`
* `TB6600 DIR-` ➔ STM32 `PB13`
* `TB6600 EN-`  ➔ STM32 `PB14` (使能)
* 拨码开关设定：**细分 800**，**电流 1.5A**。

### 3. 电磁铁接线 (MOS 模块隔离驱动)
* `MOS 控制端 PWM/SIG` ➔ STM32 `PA1`
* `MOS 控制端 VCC & GND` ➔ STM32 `3.3V & GND`
* `MOS 输出端 VIN+ & VIN-` ➔ 外部独立 **12V 电源**
* `MOS 输出端 OUT+ & OUT-` ➔ 接电磁铁线圈两端

### 4. 传感器接线 (I2C)
* `MAX30105 SCL` ➔ STM32 `PB6`
* `MAX30105 SDA` ➔ STM32 `PB7`

## 🛠️ 环境部署与操作 (How to Run)

1. **烧录 STM32**：使用 VSCode + PlatformIO 打开 `HeartRate_Device`，连接串口烧录。
2. **配置 Python 环境**：
   ```bash
   pip install torch torchvision torchaudio opencv-python pyserial numpy
   ```
3. **启动祭坛**：
   * 开启硬件 12V 总电源。
   * 运行 Python 脚本：`python final_stable_lie_detector.py`
4. **开始仪式**：将手指放置于发着红光的 MAX30105 传感器上，注视屏幕。电机与电磁铁将彻底接管你的生理节律。

## 🎛️ 终极微调指南 (Tuning & Calibration)

艺术表达需要极致的物理手感。您可以在 STM32 的 `main.cpp` 中微调以下宏定义与参数：

### 1. 穿刺控制：深渊模式与狂暴模式
* **深渊窒息模式 (极端心率惩罚)**：
  当检测到极端异常心率或无人状态时，系统会触发 `MAX_CYCLE_TIME = 30.0`。机器将以极度缓慢、极具折磨感的速度进行长达 30 秒的深渊级蠕动穿刺，带来巨大的心理压迫。
* **物理极限防堵转 (安全底线)**：
  `MIN_CYCLE_TIME = 2.5` 设定了机器最快爆发速度为 2.5 秒一周期。如果步进电机在高速时发出尖叫声并卡死（失步），请将其调高到 3.0。
* **情绪暴走倍率**：
  在 `calculateMotionParameters()` 中，如识别到愤怒：
  `if (currentEmotion == "anger") { emo_factor = 2.0; cycles = 2; }`
  把 `2.0` 改为 `3.0` 会极其狂暴；把 `cycles = 2` 改为 `4`，将持续暴走更久。

### 2. 电磁铁抽动控制 (力度与痉挛感)
* **修改通电时长**：代码常量 `SOLENOID_PULSE_MS = 100`。
  * 默认 `100ms` 可提供沉闷且有力的金属撞击感。
  * 若要极度干脆的“啪嗒”声，可降低至 `60`。
  * ⚠️ *警告：切勿超过 200ms，否则高心率下电磁铁会积热烧毁。*
* **物理调优**：如果觉得抽动力度弱，请用机械结构限制电磁铁的退回间隙。让铁芯初始位置离吸合底部只剩 5-6mm，爆发力会呈指数级增强。

### 3. 温度说明
> 步进电机在通电待机时表面温度达到 50℃-65℃ 属于**绝对正常现象**（满电流锁死特性）。
> 若使用 3D 打印的 **PLA** 材质制作电机支架，高温会导致其变软变形。请使用 PETG/ABS 打印，或在电机与支架间垫入隔热垫。
